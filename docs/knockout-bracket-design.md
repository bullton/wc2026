**日期**: 2026-01-25

**状态**: 设计完成

## 1. 需求概述

重新设计世界杯淘汰赛对阵图，采用经典树状图样式，决赛居中显示，上下两个半区从两侧向中间汇聚。

### 设计目标

- 经典树状图布局，从左到右阅读
- 决赛居中，上下分区清晰
- 使用 SVG 绘制连接线，DOM 显示比赛卡片
- 前端根据后端比赛结果自动计算晋级关系
- 保持与小组赛积分表一致的架构（前端计算逻辑）

### 轮次间距计算

1/16决赛到1/8决赛的垂直间距规律：
- 1/16决赛: 8px间距，需要覆盖16场比赛
- 1/8决赛: 30px间距，需要覆盖8场比赛
- 1/4决赛: 90px间距，需要覆盖4场比赛
- 半决赛: 210px间距，需要覆盖2场比赛
- 决赛: 居中显示

### SVG连接线设计

```javascript
// SVG连接线生成逻辑
function generateConnectorPath(startY, endY, startX, endX) {
  // 使用贝塞尔曲线实现平滑连接
  const midX = (startX + endX) / 2;
  
  // 水平线 + 垂直线 + 水平线的组合路径
  return `M ${startX} ${startY} 
          L ${midX} ${startY} 
          L ${midX} ${endY} 
          L ${endX} ${endY}`;
}


// 获胜队伍连接线样式
.winner-line {
  stroke: #28a745;
  stroke-width: 3px;
  stroke-dasharray: none;
}

// 待定比赛连接线样式

  stroke: #888;

  stroke-dasharray: 5,3;
}
```
